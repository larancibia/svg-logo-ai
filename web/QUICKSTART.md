# Quick Start - LLM-QD Web Visualization

Get the visualization running in **30 seconds**.

## View Locally

### Option 1: Python (Recommended)
```bash
cd web
python3 -m http.server 8000
```
Open: http://localhost:8000

### Option 2: Node.js
```bash
cd web
npx http-server -p 8000
```
Open: http://localhost:8000

### Option 3: Interactive Script
```bash
cd web
./deploy.sh
# Select option 5 or 6
```

## Deploy Online

### Cloudflare Pages (Fastest)
1. Push to GitHub
2. Go to https://pages.cloudflare.com
3. Connect repository
4. Build output: `web`
5. Deploy!

### GitHub Pages (Simplest)
1. Repository Settings > Pages
2. Source: Branch `main`, Folder `/web`
3. Save
4. Done!
